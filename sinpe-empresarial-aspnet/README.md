# SINPE Empresarial ASP.NET

Sistema web desarrollado como proyecto académico del curso **Programación Avanzada**, orientado a la gestión y verificación de pagos electrónicos mediante la plataforma SINPE Empresarial.

La aplicación permite a los comercios afiliados registrar y administrar sus datos, gestionar cajas asociadas a números SINPE y verificar los pagos electrónicos recibidos, facilitando la integración con sistemas internos de facturación.
